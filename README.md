# Weather App

Simple weather application built with HTML, CSS and JavaScript.

## Features

- Search weather by city name
- Current temperature
- Current wind speed
- City geocoding
- Error handling
- Loading state
- Search by Enter key

## Technologies

- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API
- Async / Await

## APIs

### Open-Meteo Weather API

Used for getting current weather data.

### Open-Meteo Geocoding API

Used for converting city names into coordinates.

## How to Use

1. Enter a city name.
2. Press **GET WEATHER** or hit **Enter**.
3. View current temperature and wind speed.
